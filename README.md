# Robinwings
A very fast website that is for viewing basic data of Honkai: Star Rail such as:
- character: id, stats, abilities, ascension materials, etc.
- lightcones: id, stats, abilities, ascension materials, etc.
- relics: id, abilities

Has a complete search & filter feature too!
